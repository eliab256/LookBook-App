# Node Globals

__dirname = percorso cartella corrente
__filename = percordo file
require = usare i moduli. è deprecated, ora si usa import {} from "";
process = info relative all' ambiente di sviluppo

## moduli built-in: os, path. 
- os recupera informazioni riguardo al sistema operativo del computer
- path serve per lavorare con i percorsi

installare nodemon come dev dependencie per non dover aggiornare ogni volta con node index. tipo live server per html

## Metodi http

GET --> leggere dati
POSR --> inviare dati
PUT --> modificare dati gia esistenti
DELETE --> eliminare dati
